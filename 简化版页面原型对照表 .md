# 舒尔特方格小程序 \- 详细页面原型文案\+布局结构

全局风格：马卡龙柔和护眼风（低饱和浅底色、大圆角元素、无刺眼对比色、字体圆润、数字随机柔和彩色）；所有按钮统一圆角设计，浅色系填充；页面间距均匀，视觉清爽不拥挤。

# 页面1：首页（核心入口页）

## 一、布局结构（从上到下，居中对称）

1. 顶部导航栏（无标题，仅左侧返回图标，右侧个人中心图标）

2. 欢迎区域（居中，占页面1/5高度）

3. 签到展示区（居中，紧接欢迎区，占页面1/10高度）

4. 难度选择区（核心区域，占页面1/2高度，网格布局）

5. 快捷入口区（底部，占页面1/10高度）

6. 游客提示条（最底部，固定显示）

## 二、详细文案\+元素说明

1. 顶部导航栏
       

    - 左侧：返回图标（仅非首次进入显示，点击返回上一页；首次进入隐藏）

    - 右侧：个人中心图标（马卡龙浅粉色，圆润线条，点击跳转个人中心页）

    - 背景：马卡龙浅米色，无阴影，简洁干净

2. 欢迎区域
        

    - 主标题：**舒尔特方格**（字体加粗，马卡龙深紫色，字号24rpx）

    - 副标题：专注力训练 · 全年龄段适用（字体常规，马卡龙浅紫色，字号16rpx，与主标题间距8rpx）

    - 背景：无额外背景，与页面整体底色一致

3. 签到展示区
        

    - 内容：连续签到 **X** 天 \| 今日已打卡/今日未打卡（字体常规，马卡龙浅橙色，字号14rpx）

    - 样式：圆角矩形（半径12rpx），浅橙色填充（低饱和），内边距8rpx 20rpx，居中显示

    - 交互：未打卡时，点击区域触发签到，签到成功后提示“签到成功，连续签到\+1”（柔和弹窗，3秒自动消失）

4. 难度选择区（核心）
        

    - 布局：2列3行网格，每个难度一个卡片，间距16rpx，整体居中

    - 卡片样式：圆角矩形（半径16rpx），马卡龙浅色系填充（3×3浅蓝、4×4浅粉、5×5浅黄、6×6浅绿、7×7浅紫、8×8浅橙），轻微阴影，内边距20rpx

        - 难度名称：入门 / 初级 / 经典 / 高级 / 专家 / 大神（字体常规，字号14rpx，浅灰色，居中，与标识间距4rpx）

        - 个人最佳（登录后显示）：最佳用时 XX\.XXX 秒（字体常规，字号12rpx，浅灰色，居中）

    - 交互：点击任意卡片，跳转至对应难度的训练页

5. 快捷入口区
        

    - 布局：2个图标\+文字，左右对称，间距40rpx

    - 入口1：排行榜（马卡龙浅绿色图标\+文字，文字字号14rpx，点击跳转排行榜页）

    - 入口2：我的记录（马卡龙浅蓝色图标\+文字，文字字号14rpx，点击跳转个人中心页）

    - 样式：固定在页面最底部，背景与页面底色一致，无额外边框

# 页面2：训练页（核心对局页）

## 一、布局结构（从上到下，重点突出方格区域）

1. 顶部信息栏（占页面1/10高度）

2. 倒计时/计时器区域（居中，占页面1/10高度）

3. 舒尔特方格区域（核心，占页面1/2高度，居中显示）

4. 操作按钮区（底部，占页面1/10高度）

5. 广告预留区（最底部，临时占位，后续接入横幅广告）

## 二、详细文案\+元素说明

1. 顶部信息栏
        

    - 左侧：返回按钮（马卡龙浅灰色图标，点击返回首页，不保存当前对局）

    - 中间：难度标识（3×3 / 4×4\.\.\.，字体加粗，字号18rpx，马卡龙深紫色）

    - 右侧：错误次数（文案：错误：0，字体常规，字号16rpx，马卡龙浅红色）

    - 背景：马卡龙浅米色，与首页一致

2. 倒计时/计时器区域
        

    - 倒计时状态（开局前3秒）：
                

        - 文案：3 → 2 → 1（字体加粗，字号32rpx，马卡龙随机柔和彩色，居中显示）

        - 背景：无，倒计时结束后自动切换为计时器

    - 计时器状态（倒计时结束后）：
                

        - 文案：XX\.XXX（秒，毫秒级，字体加粗，字号32rpx，马卡龙深紫色，居中显示）

        - 样式：计时器字体固定，不随页面缩放变形，实时刷新显示

3. 舒尔特方格区域（核心）
        

    - 布局：按选择难度展示对应网格（3×3/4×4\.\.\.），自适应屏幕大小，上下左右居中，网格间距8rpx

    - 方格样式：每个方格为圆角矩形（半径12rpx），马卡龙浅白色填充，轻微阴影，边框为对应难度的浅色系（与首页难度卡片配色一致）

    - 数字样式：每个方格内数字随机马卡龙柔和彩色（低饱和，不刺眼），字体加粗，字号随方格大小自适应，居中显示

    - 交互规则：
                

        - 按1→2→3\.\.\.顺序点击，点击正确的数字，方格轻微变色（对应数字颜色的浅色调），无震动反馈

        - 点击错误的数字，错误次数\+1，方格无变色，无弹窗提示，计时器继续计时

        - 点击完最后一个数字后，自动停止计时，跳转至成绩结算页

4. 操作按钮区
        

    - 布局：2个按钮，左右对称，间距40rpx，居中显示

    - 按钮1：重置本局（圆角矩形，半径12rpx，马卡龙浅粉色填充，文字黑色，字号16rpx，内边距12rpx 32rpx）
                

        - 交互：点击后，重置当前对局，重新显示3、2、1倒计时，错误次数清零，数字重新随机生成

    - 按钮2：退出对局（圆角矩形，半径12rpx，马卡龙浅灰色填充，文字黑色，字号16rpx，内边距12rpx 32rpx）
               

        - 交互：点击后，弹出柔和提示“确定退出对局？退出后不保存本次成绩”，点击确定返回首页，点击取消留在当前页面

5. 广告预留区
       

    - 样式：圆角矩形（半径8rpx），占位文案“广告区域”，浅灰色填充，居中显示，高度固定为80rpx

    - 后续：替换为微信横幅广告，不遮挡操作按钮，不干扰对局体验
